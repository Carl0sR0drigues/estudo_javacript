For

Estrutura de repetição

O for é uma estrutura de repetição no JavaScript usada para executar um bloco de código várias vezes.
Ele é especialmente útil quando você sabe exatamente quantas vezes deseja repetir a execução.


Estrutura Básica do for


Como Funciona?

Inicialização: Declara e inicializa uma variável de controle, executada uma única vez antes do loop começar.
Condição: Avaliada antes de cada iteração. Se for verdadeira, o bloco de código é executado. Se for falsa, o loop para.
Incremento: Atualiza a variável de controle após cada iteração.
