1-Operadores Aritimeticos

Os operadores aritimeticos são os famosos sinais matematicos:
Soma "+",
Subtração "-",
Multiplicação "*" e
Divisão "/"
Eles são usado para fazer os calculos que conhecemos no nosso dia a dia, esse sinais podem ser usados para efetuas operações de incremento e decremento.
Incremento:
valor1 = 5
total = ++valor1
retorno 6, o valor1 recebeu um incremento de mais 1.
Decremento:
valor1 = 5
total = --valor1
retorno 4, o valor1 recebeu um decremento de menos 1.

2-Operadores atribuição

Os operadores de atribuição tem a capacidade de atribuir um valor a uma variavel, como o operador de igualdade "=".
Exemplo;
var ;
valor1 = x;
tambem temos os operadores de atribuição soma (+)
Exemplo;
var valor1, valor2;
valor1 = 5
valor2 = 10
valor1 += valor2
A resposta é 15.

3-Operador Sequencial

Os operadores de sequencia são os operadores que podem juntar duas ou mais string.
Exemplo:
valor1 = "carlos ";
valor2 = "rodrigues";
total = valor1 + valor2;
retorno "carlos rodrigues"

4-Operadores comparação

Os operadores de comparação retornam apenas dois estados False ou True.
Os sinas desse operador são (==, ===, >, <, !=, !==).
Sinal de "=" é usado quando queremos atribuir um valor a uma variavel.
Sinal de "==" é usado quando queremos comparar duas ou mais coisas, não considerando os tipos.
Sinal de "===" é usado quando queremos comparar duas ou mais coisas, considerando os tipos para comparação.
Sinal de "!=" é usado quando queremos saber se uma coisa é diferente de outra, não considerando os tipos.
Sinal de "!==" é usado quando queremos saber se uma coisa é diferente de outra, considerando os tipos para comparação.
Sinal de ">" é usado quando queremos saber se uma coisa é maior que outra.
Sinal de "<" é usado quando queremos saber se uma coisa é menor que outra.
Os sinais de menor e maior podem ser combinados com o sinal de igual, para saber se é maior ou igual e menor ou igual.

Exemplo:
valor1 = 2
valor2 = 4
res = (valor1 == valor2)
retorno "False"

valor1 = 5
valor2 = 5
res = (valor1 == valor2)
retorno "True"

5-Operador Condicional

Os operadores condicionais são dependentes de algum fator pré determinado.
Exemplo:
var idade, eleitor;
idade = 20;
eleitor = (idade < 18) ? "Não, Eleitor" : "Sim, eleitor";
alert("A resposta é: " + eleitor + "a idade dele é de: " + idade);

Neste caso o operador é toda a linha eleitor que esta condicionado a idade, o alert é apenas uma forma de apresentar
o retorno do programa, para que já conhece o "if" ele é muito parecido

idade = 20 Condicão
eleitor = (idade < 18) ? Operador Condicional
"Não, Eleitor" retorno caso a condicão seja True
: ou
"Sim, eleitor" retorno caso a condição seja False

6-Operadores Logicos

Os operadores logicos são muito parecidos com os operadores condicionais, porem tendo mais de uma fator pré determinado.
Exemplo:
var idade, resultado;
idade = 20;
resultado = (idade > 60 && idade < 70);
Neste caso o operador é toda a linha resultado que esta condicionado a idade, sendo usado o "&&" para representar o "e", "idade > 60 e idade < 70"

resultado = (idade > 60 || idade < 70);
Aqui o operador que muda é "||" que representa "ou", "idade > 60 ou idade < 70"

resultado = !(idade > 60 || idade < 70);
Aqui o operador que muda tudo é "!" que representa "negação", "não é (idade > 60 ou idade < 70)"
Em termos simples, a negação vai considerar True se a condicão for False e False se a condição for True
