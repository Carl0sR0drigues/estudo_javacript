EVENTOS DE TEMPO COM JAVASCRIPT

Os eventos de tempo permitem a execução do código em intervalos de tempo especificados.Esses intervalos de tempo são chamados de eventos de cronometragem.

Os dois métodos-chave para usar com JavaScript são:

setTimeout(function, tempo em milisegundos)
Executa uma função, depois de esperar um número especificado de milissegundos.
setInterval(function, milliseconds) É o mesmo que setTimeout(), mas repete a execução da função continuamente.

CODIGO DAS FUNÇÕES ACIMA


Esse código contém funções para gerenciar dois tipos de temporizadores em JavaScript: setTimeout e setInterval, usados para criar temporizações ou intervalos periódicos. Aqui está uma explicação detalhada de cada função:

1. function ativar()

Propósito: Aguarda 3 segundos antes de executar uma ação.

Funcionamento:

  • Muda o texto do elemento com o ID "tempo" para "Aguarde 3 segundos".
  • Usa setTimeout para definir uma ação a ser executada após 3000 milissegundos (3 segundos):
    • Altera o texto do elemento para "Executou o setTimeout".
    • Muda a cor de fundo da página para verde.
  • A variável tempo armazena a referência ao temporizador, permitindo que ele seja cancelado.

2. function desativar()

Propósito: Cancela o temporizador criado com setTimeout.

Funcionamento:

  • Usa clearTimeout(tempo) para cancelar o temporizador armazenado na variável tempo.
  • Altera o texto do elemento com o ID "tempo" para "Contagem parada".

3. function contar()

Propósito: Inicia um contador que incrementa o valor exibido na página a cada segundo.

Funcionamento:

  • Usa setInterval para executar um bloco de código a cada 1000 milissegundos (1 segundo).
  • No intervalo:
    • Obtém o valor atual do elemento com o ID "temporismo".
    • Converte o valor para inteiro com parseInt e incrementa-o.
    • Atualiza o texto do elemento com o novo valor.
  • Verifica se o valor (soma) chegou a 3:
    • Reproduz um som utilizando o elemento <audio> com ID "alarme".
    • Para o intervalo com clearInterval(temporizador).

4. function parar()

Propósito: Para o intervalo criado com setInterval.

Funcionamento:

  • Usa clearInterval(temporizador) para cancelar o intervalo armazenado na variável temporizador.

Cenário de Uso

  • ativar(): Mostra uma mensagem indicando uma espera de 3 segundos e altera o fundo da página após esse período.
  • desativar(): Cancela o temporizador do setTimeout antes que ele altere a mensagem ou o fundo.
  • contar(): Inicia um contador que incrementa o número exibido no elemento temporismo. Quando o número atinge 3:
    • Toca um som.
    • Para o contador.
  • parar(): Interrompe o contador a qualquer momento, independentemente do valor atual.

Resumo:

  • setTimeout: Executa uma ação única após um tempo determinado.
  • setInterval: Executa uma ação repetidamente em intervalos definidos.
  • clearTimeout e clearInterval: Cancelam temporizadores ou intervalos em execução.

Essas funções são fundamentais para gerenciar eventos com temporização no JavaScript!
