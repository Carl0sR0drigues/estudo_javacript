Operadores atribuição

Os operadores de atribuição tem a capacidade de atribuir um valor a uma variavel, como o operador de igualdade "=".
Exemplo;
var ;
valor1 = x;
tambem temos os operadores de atribuição soma (+)
Exemplo;
var valor1, valor2;
valor1 = 5
valor2 = 10
valor1 += valor2
A resposta é 15.
Isso é o mesmo que valor1 = valor1 + valor2 isso tambem vale para os sinais de (-=, *=, /=).
